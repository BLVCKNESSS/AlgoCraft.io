Présentation

Cette présentation illustre comment créer un algorithme simple dans AlgoCraft, permettant de calculer l’image d’un réel x par la fonction linéaire f(x) = 3x + 4. Ce cas pratique vous guidera pas à pas dans l’utilisation des principales fonctionnalités de l’application.

  1. Déclaration des variables : Commencez par déclarer la variable x en cliquant sur "Déclarer nouvelle variable" dans l’interface. Saisissez "x" dans le champ dédié au nom, puis sélectionnez le type adapté : entier pour les nombres entiers ou réel pour des nombres décimaux. Validez pour enregistrer la variable.
  2. Répétez l’opération pour déclarer une seconde variable y qui contiendra le résultat de l’expression mathématique.
  3. Positionnement dans le code : Assurez-vous que le curseur ou la sélection dans l’éditeur AlgoCraft est positionné sur la ligne d’instruction DEBUT_ALGORITHME. Cela garantit que les instructions que vous ajouterez seront bien prises en compte dans l’exécution de votre algorithme.
  4. Ajout d’instructions : Cliquez sur "Nouvelle Ligne" pour insérer une ligne d’instruction vide dans l’éditeur.
  5. Lecture de la variable : Pour demander la valeur de x à l’utilisateur lors de l’exécution, utilisez le bouton "Ajouter LIRE variable". Sélectionnez la variable x dans la liste proposée et validez. Cette instruction permet à l’algorithme d’attendre une entrée utilisateur.
  6. Affectation d’une expression : Placez le curseur sur la ligne suivante, cliquez sur "Nouvelle Ligne" puis "AFFECTER valeur à variable". Choisissez y comme variable cible, puis saisissez l’expression 3*x + 4 dans le champ correspondant. Validez pour enregistrer l’affectation.
  7. Affichage du résultat : Après l’instruction d’affectation, ajoutez une nouvelle ligne. Cliquez sur "Ajouter AFFICHER variable", sélectionnez la variable y et validez. Cette étape affichera la valeur calculée dans la console intégrée lors de l’exécution.
  8. Test de l’algorithme : Cliquez sur le bouton "Tester" pour ouvrir la fenêtre de test d’exécution. Dans cette nouvelle fenêtre, cliquez sur "Lancer l’algorithme". Une invite vous demandera de saisir une valeur pour x. Après saisie, le résultat calculé (l’image de la fonction) s’affichera dans la zone "Résultat". Vous pouvez relancer le calcul autant de fois que souhaité sans fermer la fenêtre.
  9. Pour revenir à la fenêtre principale, cliquez simplement sur "Fermer". Vous pouvez ensuite sauvegarder, modifier ou étendre votre algorithme selon vos besoins.

Caractéristiques

AlgoCraft est une plateforme conçue pour faciliter la création, l’édition, et l’exécution d’algorithmes sous forme textuelle, tout en offrant une interface utilisateur claire et interactive. Voici ses principales caractéristiques :

  • Déclaration simple et souple des variables : Les utilisateurs peuvent créer des variables de différents types (entier, réel, booléen, char) via une interface graphique intuitive, sans nécessiter de syntaxe complexe.
  • Éditeur textuel intégré : AlgoCraft propose un éditeur où les algorithmes sont écrits sous forme de texte libre structuré, avec des instructions simples et lisibles, facilitant la compréhension et la modification.
  • Interpréteur dynamique : Les instructions sont analysées et exécutées à la volée par un interpréteur intégré, permettant une exécution interactive et rapide sans compilation préalable.
  • Console asynchrone : L’application inclut une console capable de lire des entrées utilisateur de manière asynchrone et d’afficher les résultats ou messages en temps réel, améliorant ainsi l’interactivité.
  • Fonctions dynamiques : Possibilité de créer des fonctions personnalisées avec un corps textuel et un type de retour, enregistrées et utilisées dynamiquement dans les algorithmes. Cette version ne supporte pas encore les paramètres, mais la modularité est assurée.
  • Validation et débogage : AlgoCraft offre des mécanismes pour détecter des erreurs de syntaxe ou des incohérences avant exécution, réduisant les risques de plantage et facilitant le développement.
  • Modularisation : Les algorithmes peuvent être organisés en fonctions, permettant de structurer et simplifier la maintenance et l’évolution des projets complexes.
  • Extensibilité : Le système prévoit d’étendre les fonctionnalités, notamment la gestion des paramètres de fonctions, la surcharge, la visibilité (publique/privée), et d’autres concepts avancés de programmation.
  • Interopérabilité et gestion des fichiers : L’application peut ouvrir et sauvegarder des fichiers au format .agc, avec association automatique au système via le registre Windows, facilitant le lancement direct depuis l’explorateur.
  • Utilisation de bibliothèques mathématiques : AlgoCraft utilise des bibliothèques comme MathNet.Numerics pour le traitement mathématique avancé, garantissant précision et performance.
