1. Introduction

Qu'est-ce qu'AlgoCraft ?

AlgoCraft 2.0 est un logiciel libre et multi-plateforme d'aide à l'élaboration et à l'exécution d'algorithmes. Il offre une interface graphique intuitive pour :

  • ✍️ Écrire des algorithmes en pseudo-code
  • ▶️ Exécuter et tester votre code
  • 🎨 Bénéficier de la coloration syntaxique
  • 💡 Utiliser l'auto-complétion intelligente
  • 📤 Partager votre travail facilement

Version

  • Version actuelle : 2.0.0
  • Développé par : Un groupe d'étudiants
  • Avec la collaboration de : M. NDJE MAN
2. Installation et Démarrage

Configuration Minimale Requise

  • Système d'exploitation : Windows 7/8/10/11
  • Framework : .NET Framework 4.7.2 ou supérieur
  • Mémoire RAM : 2 GB minimum
  • Espace disque : 50 MB

Premier Lancement

  1. Lancez AlgoCraft.exe
  2. L'interface principale s'affiche avec un squelette d'algorithme
  3. Vous êtes prêt à coder ! 🚀

4. Gestion des Projets

Créer un Nouveau Projet

Menu : Fichier → Nouveau ou bouton 📁 Nouveau

  1. Nom du projet : Donnez un nom significatif
  2. Emplacement : Choisissez le dossier de sauvegarde
  3. Description : (Optionnel) Décrivez votre projet
  4. Cliquez sur Créer

Le projet est sauvegardé au format .agc (AlgoCraft)

Ouvrir un Projet Existant

Menu : Fichier → Ouvrir

  1. Naviguez jusqu'au fichier .agc
  2. Sélectionnez le fichier
  3. Cliquez sur Ouvrir

Enregistrer

  • Enregistrer : Ctrl + S ou Fichier → Enregistrer
  • Enregistrer sous : Fichier → Enregistrer sous...
5. Écriture d'Algorithmes

Structure de Base

Algorithme nom_de_l_algorithme
Variables
    // Déclarations des variables
    
Debut
    // Instructions de l'algorithme
    
Fin

Règles de Syntaxe

✅ À FAIRE :

  • Commencer par Algorithme nom
  • Déclarer les variables avant Debut
  • Terminer par Fin
  • Respecter la casse (majuscules/minuscules)

❌ À ÉVITER :

  • Oublier les mots-clés Debut et Fin
  • Utiliser des variables non déclarées
  • Oublier les guillemets pour les chaînes
6. Structures de Contrôle

🔀 Condition Si/Sinon

Si condition Alors
    // Instructions si vrai
Sinon
    // Instructions si faux
FinSi

Exemple :

Si age >= 18 Alors
    Ecrire("Majeur")
Sinon
    Ecrire("Mineur")
FinSi

🔁 Boucle Pour

Pour variable <- debut à fin Faire
    // Instructions à répéter
FinPour

Exemple :

Pour i <- 1 à 10 Faire
    Ecrire("Nombre : ", i)
FinPour

🔄 Boucle Tant Que

Tant que condition Faire
    // Instructions à répéter
FinTantQue

Exemple :

Tant que x < 100 Faire
    x <- x * 2
    Ecrire(x)
FinTantQue

🔃 Boucle Répéter

Repeter
    // Instructions à répéter
Jusqu'a condition

Exemple :

Repeter
    Lire(nombre)
    Ecrire("Vous avez saisi : ", nombre)
Jusqu'a nombre = 0
7. Fonctions Mathématiques

📐 Fonctions à 1 Paramètre

Fonction Syntaxe Description Exemple

Racine Racine(x) Racine carrée Racine(16) → 4
Cos Cos(x) Cosinus (radians) Cos(0) → 1
Sin Sin(x) Sinus (radians) Sin(0) → 0
Tan Tan(x) Tangente (radians) Tan(0) → 0

🔢 Fonctions à 2 Paramètres

Fonction Syntaxe Description Exemple

Minimum Minimum(x, y) Plus petit nombre Minimum(10, 20) → 10
Puissance Puissance(x, y) x élevé à la puissance y Puissance(2, 3) → 8
8. Variables et Types

Déclaration de Variables

Variables
    nom_variable : type

Types Disponibles

Type Description Exemple de Valeur
entier Nombre entier 42, -17, 0
reel Nombre décimal 3.14, -0.5, 2.0
chaine Texte "Bonjour", "AlgoCraft"

Affectation

variable <- valeur

Exemples :

Variables
    age : entier
    prix : reel
    nom : chaine
    
Debut
    age <- 25
    prix <- 19.99
    nom <- "Alice"
Fin
9. Exécution et Débogage

Exécuter un Programme

  1. Écrivez votre algorithme
  2. Cliquez sur ▶️ Tester ou Menu → Exécuter
  3. Consultez les résultats dans la Console
  4. Vérifiez les éventuelles erreurs dans la Zone d'erreurs

Lecture de Données

Lire(variable)

Une boîte de dialogue s'affiche pour saisir la valeur.

Affichage de Données

Ecrire(element1, element2, ...)

Exemples :

Ecrire("Bonjour")
Ecrire("Résultat : ", x)
Ecrire("x = ", x, " et y = ", y)

Comprendre les Erreurs

  • ❌ Variable non déclarée → Déclarez la variable dans la section Variables
  • ❌ FinSi manquant → Ajoutez FinSi après votre bloc conditionnel
  • ❌ Syntaxe invalide → Vérifiez la structure : Si condition Alors

13. Exemples Pratiques

🔢 Exemple 1 : Calcul de Moyenne

Algorithme calcul_moyenne
Variables
    note1 : reel
    note2 : reel
    note3 : reel
    moyenne : reel
    
Debut
    Ecrire("=== CALCUL DE MOYENNE ===")
    
    Ecrire("Entrez la note 1 :")
    Lire(note1)
    
    Ecrire("Entrez la note 2 :")
    Lire(note2)
    
    Ecrire("Entrez la note 3 :")
    Lire(note3)
    
    moyenne <- (note1 + note2 + note3) / 3
    
    Ecrire("Moyenne : ", moyenne)
    
    Si moyenne >= 10 Alors
        Ecrire("ADMIS")
    Sinon
        Ecrire("RECALÉ")
    FinSi
Fin

🔢 Exemple 2 : Table de Multiplication

Algorithme table_multiplication
Variables
    nombre : entier
    i : entier
    
Debut
    Ecrire("Table de multiplication")
    Ecrire("Entrez un nombre :")
    Lire(nombre)
    
    Ecrire("===================")
    Pour i <- 1 à 10 Faire
        Ecrire(nombre, " x ", i, " = ", nombre * i)
    FinPour
Fin

🎲 Exemple 3 : Deviner un Nombre

Algorithme deviner_nombre
Variables
    secret : entier
    essai : entier
    compteur : entier
    
Debut
    secret <- 42
    compteur <- 0
    
    Ecrire("=== JEU : DEVINEZ LE NOMBRE ===")
    Ecrire("J'ai choisi un nombre entre 1 et 100")
    
    Repeter
        Ecrire("Votre essai :")
        Lire(essai)
        compteur <- compteur + 1
        
        Si essai < secret Alors
            Ecrire("Trop petit !")
        FinSi
        
        Si essai > secret Alors
            Ecrire("Trop grand !")
        FinSi
    Jusqu'a essai = secret
    
    Ecrire("BRAVO ! Trouvé en ", compteur, " essais !")
Fin

📐 Exemple 4 : Calculs Géométriques

Algorithme geometrie
Variables
    rayon : reel
    cote : reel
    perimetre : reel
    aire : reel
    pi : reel
    
Debut
    pi <- 3.14159
    
    // Cercle
    Ecrire("=== CERCLE ===")
    Ecrire("Rayon du cercle :")
    Lire(rayon)
    
    perimetre <- 2 * pi * rayon
    aire <- pi * Puissance(rayon, 2)
    
    Ecrire("Périmètre : ", perimetre)
    Ecrire("Aire : ", aire)
    
    // Carré
    Ecrire("")
    Ecrire("=== CARRÉ ===")
    Ecrire("Côté du carré :")
    Lire(cote)
    
    perimetre <- 4 * cote
    aire <- Puissance(cote, 2)
    
    Ecrire("Périmètre : ", perimetre)
    Ecrire("Aire : ", aire)
Fin

🧮 Exemple 5 : Fonctions Mathématiques

Algorithme test_fonctions_math
Variables
    a : entier
    b : entier
    resultat : reel
    
Debut
    a <- 16
    b <- 9
    
    // Racine carrée
    resultat <- Racine(a) + Racine(b)
    Ecrire("Racine(16) + Racine(9) = ", resultat)
    
    // Puissance
    resultat <- Puissance(2, 5)
    Ecrire("2^5 = ", resultat)
    
    // Maximum
    resultat <- Maximum(Absolu(-10), Absolu(7))
    Ecrire("Maximum(|-10|, |7|) = ", resultat)
    
    // Minimum
    resultat <- Minimum(25, 30)
    Ecrire("Minimum(25, 30) = ", resultat)
Fin
14. Dépannage

Problèmes Courants

❓ L'application ne démarre pas

  • Vérifiez que .NET Framework 4.7.2+ est installé
  • Exécutez en tant qu'administrateur
  • Vérifiez les fichiers d'installation

❓ Le code ne s'exécute pas

  • Vérifiez la structure : Algorithme ... Debut ... Fin
  • Assurez-vous que toutes les variables sont déclarées
  • Regardez les messages d'erreur détaillés

❓ Les fonctions mathématiques ne fonctionnent pas

  • Vérifiez la syntaxe : Maximum(x, y) avec virgule
  • Assurez-vous que les paramètres sont numériques
  • Utilisez des variables déclarées

❓ L'auto-complétion ne s'affiche pas

  • Vérifiez la connexion à la base de données
  • Tapez quelques caractères pour déclencher l'auto-complétion
  • Redémarrez l'application

❓ Impossible de sauvegarder

  • Vérifiez les droits d'accès au dossier
  • Assurez-vous que le disque n'est pas plein
  • Utilisez un chemin sans caractères spéciaux

❓ Erreurs de syntaxe fréquentes

Erreur Solution
Variable 'x' non déclarée Ajoutez x : type dans la section Variables
FinSi manquant Terminez chaque Si par FinSi
FinPour manquant Terminez chaque Pour par FinPour
FinTantQue manquant Terminez chaque Tant que par FinTantQue
Syntaxe invalide pour Si Utilisez : Si condition Alors
